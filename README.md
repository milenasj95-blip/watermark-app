# watermark-app
Python application with a graphical interface to apply watermarks to images.
Aplicação de Marca d’Água em Imagens
Descrição

Aplicação desenvolvida em Python que permite aplicar automaticamente uma marca d’água em imagens, por meio de uma interface gráfica simples e intuitiva.
O projeto foi criado com foco em automação, organização de código e usabilidade.

Funcionalidades:

Seleção de imagens do computador

Aplicação de marca d’água em imagens

Ajuste de posição e/ou opacidade da marca d’água

Salvamento da imagem final com a marca aplicada

Interface gráfica amigável para facilitar o uso

Tecnologias Utilizadas

Python

Pillow (processamento de imagens)

Tkinter (ou PyQt, se for o caso)

Como Executar o Projeto

Clone este repositório:

git clone https://github.com/seu-usuario/watermark-app.git


Acesse a pasta do projeto:

cd watermark-app


Instale as dependências:

pip install -r requirements.txt


Execute a aplicação:

python main.py

Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento em Python, integração entre lógica de negócio e interface gráfica, além de servir como projeto de portfólio para oportunidades na área de desenvolvimento de software.
